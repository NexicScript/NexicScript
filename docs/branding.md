# NexicScript Brand Guide

This guide defines the public brand identity for the NexicScript organization.

## Visual identity

NexicScript uses a dark blue and purple palette.

- Dark blue: `#0F172A`
- Purple: `#7C3AED`

## Official logo

Use [`branding/Logo.png`](../branding/Logo.png) as the official NexicScript logo.

This is the logo that should be used in the public README, release assets, and organization pages.

## Banner

Use [`branding/banner.png`](../branding/banner.png) as the official banner image.

## Recommended usage

- Keep the logo proportions intact.
- Use clear spacing around the logo.
- Do not stretch, rotate, or recolor the logo unless a separate brand asset is created for that purpose.
- Use the logo consistently across npm pages, GitHub pages, and docs.
